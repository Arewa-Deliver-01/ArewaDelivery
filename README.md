# ArewaDeliver – Multi-Role Delivery Platform

ArewaDeliver is a modern, mobile-first delivery platform designed for:
- **Customers** (request deliveries / track orders)  
- **Vendors** (manage orders & products)  
- **Riders** (accept and deliver tasks)  
- **Admins** (manage the ecosystem)

This project includes:
- 🔹 Complete HTML UI (Customer / Vendor / Rider / Admin)
- 🔹 Tailwind-based modern UI design
- 🔹 Clean folder structure for deployment
- 🔹 Centralized global CSS and JS files
- 🔹 Ready for backend integration (Firebase / Node / APIs)
